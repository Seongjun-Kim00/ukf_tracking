# ukf_tracking
Unsecented Kalman Filter(UKF) based LiDAR tracking

This tracking module is using LiDAR detection(,whose consists of relative postion, yaw, class and etc).

# Modules
- Association_dets_to_trks.py

- calculatedBevIOU.py

- parameter_ukf.py

- ros_topic_tool.py

- sort_lidar.py

- track_management.py

- ukf_box_tracker.py
